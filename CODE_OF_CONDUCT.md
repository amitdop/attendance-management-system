# 📋 Code of Conduct

## Our Pledge

We are committed to providing a welcoming, inclusive, and harassment-free experience for all contributors, regardless of age, body size, disability, ethnicity, gender identity, level of experience, nationality, personal appearance, race, religion, sexual identity, or sexual orientation.

## Our Standards

Examples of behavior that creates a positive environment include:

✅ **DO:**
- Be respectful and inclusive
- Use welcoming and inclusive language
- Be patient and understanding
- Focus on constructive feedback
- Respect differing opinions and experiences
- Collaborate and support fellow contributors
- Take responsibility for mistakes
- Create a safe space for all

❌ **DON'T:**
- Use harassing, demeaning, or discriminatory language
- Make unwelcome sexual advances or comments
- Engage in personal attacks
- Post inflammatory or derogatory comments
- Publish others' private information without permission
- Engage in any form of harassment or discrimination
- Disrupt discussions or be deliberately disruptive
- Bully or intimidate other contributors

## Enforcement

### Reporting Violations

If you experience or witness unacceptable behavior, please report it by:
- Contacting the project maintainers privately
- Documenting the incident with context
- Providing relevant information for investigation

**All reports will be handled with discretion and confidentiality.**

### Consequences

Violations of this Code of Conduct may result in:
- Warning
- Temporary suspension from project activities
- Permanent removal from the project
- Legal action if applicable

## Scope

This Code of Conduct applies to:
- All project spaces (GitHub, discussions, issues, pull requests)
- Public spaces where individuals represent the project
- Private communication on behalf of the project

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.0.

## Contact

For questions or to report violations, contact the project maintainers at the GitHub repository.
